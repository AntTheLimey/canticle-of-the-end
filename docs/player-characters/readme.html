README

This folder holds optional expanded material for player characters, supplementing the entity files in Characters/PCs/. It is not a mandatory parallel structure. A PC has an entry here only when the player chooses to maintain expanded material.

File naming convention

  • <Name>_Sheet.md — detailed character sheet (stats, skills, equipment, derived values)
  • <Name>_Background.md — backstory, motivations, relationships beyond campaign cast, roleplay notes

Future entries

Any active PC whose player wants to maintain this level of detail can add entries here using the same naming convention. The pattern is opt-in, not mandatory.

What lives where

Content Location
PC entity frontmatter (type, status, relationships, tags) Characters/PCs/<Name>.md
Campaign role summary, key traits, narrative function Characters/PCs/<Name>.md
Detailed stat block, skill percentages, equipment list Player Characters/<Name>_Sheet.md (optional)
Extended backstory, personal relationships, roleplay notes Player Characters/<Name>_Background.md (optional)

See also

  • Characters/PCs/ — one entity file per PC (active, retired, deceased)
